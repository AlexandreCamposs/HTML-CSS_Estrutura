Como visualizar a fonte desta página

Se você for ao menu Exibir do navegador da Web, verá uma opção que permite ver o código-fonte por trás de uma página da Web.

Quando a web estava decolando, esta era uma maneira muito popular para as pessoas aprenderem como as páginas da web foram construídas.

Hoje, as pessoas ainda costumam ver a fonte das páginas para saber como uma página da web foi construída.
